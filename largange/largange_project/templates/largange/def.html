Interpolation de Lagrange

L'interpolation de Lagrange est une méthode mathématique qui permet de trouver un polynôme unique (appelé polynôme de Lagrange) passant exactement par une série de points donnés :

  • (x0, y0), (x1, y1), …, (xn, yn)

où les xi sont tous distincts.

Ce polynôme P(x) est de degré au plus n et s'exprime comme une combinaison linéaire des polynômes de base de Lagrange Li(x) :

P(x) = ∑i=0n yi · Li(x)

où

Li(x) = ∏j=0, j≠in   (x - xj) / (xi - xj)

Conditions nécessaires

  • Les abscisses xi doivent être distinctes (pas de doublons).
  • Le nombre de points détermine le degré du polynôme (n points → degré ≤ n-1).
